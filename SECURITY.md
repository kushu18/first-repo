# Security & Privacy Policy

## Data Visibility

This is a **public GitHub repository**. That means:

- All files, commits, issues, pull requests, and comments you post here are **visible to everyone on the internet**.
- Do **not** share any personal information, passwords, API keys, tokens, or other sensitive data in this repository.

## Chat / Issues / Discussions

If you use GitHub Issues, Pull Requests, or Discussions to communicate in this repository:

- Your messages are **public** and can be read by anyone.
- GitHub stores all content as described in [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## No Private Chat

This repository does **not** provide any private or encrypted chat feature. If you need a private conversation, please use a private channel (e.g. private email or direct message) outside of GitHub.

## Data Leak Prevention

To avoid accidental data leaks:

- Never commit secrets (passwords, API tokens, private keys) to this or any public repository.
- Review your changes with `git diff` before committing.
- If you accidentally expose sensitive data, rotate your credentials immediately and consider using [GitHub's secret scanning](https://docs.github.com/en/code-security/secret-scanning/about-secret-scanning) to detect leaks.

## Reporting a Security Issue

If you discover a security vulnerability related to this project, please **do not** open a public GitHub Issue. Instead, use one of the following private channels:

- [GitHub Private Security Advisory](https://github.com/kushu18/first-repo/security/advisories/new) — report a vulnerability privately through GitHub.
- Contact the repository owner directly via their GitHub profile.
